# front_stock

## 下载所需包执行以下在front_stock中执行以下命令
```
npm install
```

### 执行以下命令启动网页
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
